**Trong trang này**, mình sẽ chia sẻ toàn bộ hành trình thực tập AWS Cloud kéo dài **13 tuần**, từ giai đoạn học kiến thức nền tảng đến triển khai dự án **Serverless Student Management System** theo mô hình thực chiến.

Trong suốt quá trình học, mình đã trải qua:

- Tuần 1–7: Học tập kiến thức AWS nền tảng (Compute, Storage, Networking, Security, Monitoring…)
- Tuần 8: Thi Mid-term và chọn topic dự án
- Tuần 9–13: Xây dựng Backend cho hệ thống Serverless Student Management System

Dưới đây là tóm tắt công việc từng tuần:

**Tuần 1:** [Onboarding, AWS basics, IAM](1.1-week1/)

**Tuần 2:** [Security (KMS, Secrets Manager), VPC, EC2](1.2-week2/)

**Tuần 3:** [Resilient Architectures, Load Balancing, Auto Scaling, Route 53](1.3-week3/)

**Tuần 4:** [Storage (S3, EFS), Database (RDS, DynamoDB), Caching](1.4-week4/)

**Tuần 5:** [Serverless (Lambda, API Gateway), ECS/Fargate, CloudWatch](1.5-week5/)

**Tuần 6:** [Advanced Security (GuardDuty, WAF), Cost Optimization, Organizations](1.6-week6/)

**Tuần 7:** [Ôn tập tổng hợp Mid-term](1.7-week7/)

**Tuần 8:** [Thi Mid-term & chọn project: Serverless Student Management System](1.8-week8/)

**Tuần 9:** [Architecture design, DynamoDB, Cognito, Authentication APIs](1.9-week9/)

**Tuần 10:** [APIs (Classes, Subjects), Lambda deployment, API Gateway](1.10-week10/)

**Tuần 11:** [CloudWatch monitoring, Notifications, Optimization](1.11-week11/)

**Tuần 12:** [WAF, GuardDuty, CI/CD Pipeline, CloudFront](1.12-week12/)

**Tuần 13:** [Testing, Cost Optimization, Documentation, Presentation](1.13-week13/)
